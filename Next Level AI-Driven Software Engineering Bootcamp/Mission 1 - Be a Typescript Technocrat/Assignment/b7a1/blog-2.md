# Question-2. How do Pick and Omit utility types prevent code duplication while creating specialized "slices" of a master interface? Discuss how this keeps your code DRY (Don't Repeat Yourself).

- Answer-2: জেনেরিক্স (Generics) একটি ফাংশন বা কম্পোনেন্টকে বিভিন্ন ধরনের ডেটার জন্য ব্যবহার করা যায়। বারবার আলাদা টাইপের জন্য একই কোড লিখতে হবে না, ফলে কোড DRY (Don't Repeat Yourself) থাকবে।
