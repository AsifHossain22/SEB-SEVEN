# Stream & Buffer

- Stream: It uses to process a data piece by piece which is called Stream.
- Buffer: When we send data to store for a temporary time which is called Buffer.

* It is better in terms of user experience.
* Needs short memory storage as it do not complete whole process at once.

- Different types of Streams:
  1. Readable Stream - A Stream where we can read data.
  2. Writeable Stream - A Stream where we can write data.
  3. Duplex Stream - A Stream for both read and write data.
  4. Transform Stream - A Stream where we can reshape data.

- Modular System
