# Project Set Up

- Step 1. Install Typescript: Globally 'npm install -g typescript' to get access
  'tsc' command
- Step 2. Create Typescript config file: 'tsc -init'
- Step 3. Install Node as Development Dependencies: 'npm install -D @types/node'
- Step 4. Typescript config file:
  - types:["node"]
  - moduleResolution: "bundler" (to use ES and CommonJS together)
  - module: "esnext"
- Step 5. Install: NPM 'tsx' - npm install -d tsx
