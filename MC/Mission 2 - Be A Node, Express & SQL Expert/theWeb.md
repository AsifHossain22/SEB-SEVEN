# The Web - Where Everything Began

1. HTML - Building Core Structure
2. CSS - Coloring Experience (Styles)
3. JavaScript - Creating UI Interactions

- Why JavaScript invented?
  : Browser interactivity building to smooth user experience

- Why Node.js build?
  : Backend runtime environment and we can run JavaScript, V8 Engine and Node.js together in Server.

# Node.js Modules

    1. Operating System Module (os)
    2. File System Module (fs)
    3. Path Module (path)
    4. HTTP Module (http)
    5. URL Module (url)
    6. Utilities Module (util)

# Why Node.js is Popular?

    1. We can use JavaScript on server side
    2. Build highly scalable backend application
    3. It is single threaded, event driven and works non blocking I/O (input / output)
    4. Perfect building data intensive, streaming application
